# Hack Assembler

A **Hack Assembly** to **binary machine code** assembler. Type or paste Hack assembly, click **Assemble**, and get 16-bit machine code. Built with TypeScript and a simple web UI.

---

## Features

- **Assemble** — Convert Hack assembly to machine code (two-pass: symbols then instructions).
- **Sample programs** — Load Add, Max, Multiplication, or Rectangle examples with one click.
- **Clear** — Reset the editor and output so you can start fresh.
- **Error reporting** — Invalid assembly shows clear error messages (duplicate labels, illegal instructions, symbol overflow, etc.).
- **Auto-scroll** — After assembling, the page scrolls to the machine code or error section.

---

## Tech Stack

- **TypeScript** — Core assembler and UI logic.
- **Vanilla HTML / CSS / JS** — No framework; the app runs in the browser with ES modules.

## Project Structure

```
├── src/
│   ├── assembler.ts      # Two-pass assembler (symbols + code generation)
│   ├── assembler-ui.ts   # UI: samples, Assemble/Clear, output/error display
│   ├── ISA.ts            # Hack C-instruction encodings (comp, dest, jump)
│   ├── symbol-table.ts   # Predefined symbols (R0–R15, SP, LCL, SCREEN, etc.)
│   ├── ignores.ts        # Lines to ignore (comments, whitespace)
│   ├── types.ts          # TypeScript types for the ISA
│   ├── index.ts          # Entry point: initializes AssemblerUI
│   └── _test_/           # Sample assembly programs (sum, max, multi, rectangle)
├── index.html
├── style.css
├── dist/                 # Compiled JS (generated by npm run build)
└── package.json
```

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or later)
- npm (comes with Node)

### Install & build

```bash
git clone <your-repo-url>
cd "Unit 6 - Project (Assembler)"
npm install
npm run build
```

### Run the app

The app is a static front end that loads ES modules. Open it through a **local web server** (not by opening `index.html` as a file, or modules may be blocked).

**Option 1 — npx serve**

```bash
npx serve .
```

Then open the URL it prints (e.g. `http://localhost:3000`).

**Option 2 — VS Code / Cursor**

Use the **Live Server** (or similar) extension and “Open with Live Server” on `index.html`.

## Usage

1. **Enter assembly** — Type or paste Hack assembly in the textarea, or click a sample (Add, Max, Multiplication, Rectangle) to load an example.
2. **Assemble** — Click **Assemble** to run the two-pass assembler. Machine code appears in the “Machine Code” section, or an error message if the assembly is invalid.
3. **Clear** — Click **Clear** to empty the editor and hide the previous output so you can enter new code.

Supported assembly:

- **A-instructions** — `@value` or `@symbol` (numbers and symbols, including user-defined variables).
- **C-instructions** — `dest=comp;jump` (e.g. `D=M+1`, `0;JMP`).
- **Labels** — `(LABEL)` for jump targets.
- **Comments** — Lines starting with `//` are ignored.
